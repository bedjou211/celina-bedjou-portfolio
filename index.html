🚀 Projets

🚆 Vigilance Numérique (SNCF 🥇)

Problème : Détection tardive des incidents ferroviaires impactant les usagers.

Solution : Système IA de détection proactive en temps réel.

  • Architecture multi-agents (7 modules IA spécialisés)
  • Ingestion temps réel (GTFS-RT, SIRI Lite APIs)
  • Détection d’anomalies sur séries temporelles
  • Scoring intelligent (gravité + impact + confiance)
  • Génération automatique de rapports via LLM
  • Dashboard Streamlit pour monitoring

Techniques : Z-score dynamique, trend analysis, persistance temporelle

🤖 AI Customer Support System (RAG)

Problème : Support client non intelligent et non contextuel.

Solution : Système IA de réponse automatisée basé sur le contexte.

  • Pipeline RAG (Retrieval Augmented Generation)
  • Base vectorielle d’embeddings
  • Recherche sémantique (cosine similarity)
  • Classification des tickets (urgence, type, intention)
  • Génération de réponses via LLM

Résultat : Réduction du temps de traitement et amélioration des réponses.
