🚀 Projets

🚆 Vigilance Numérique (SNCF Hackathon – 🥇 1ère place)

Objectif : Détection précoce des perturbations ferroviaires en temps réel à partir de flux de données massifs.

  • Conception d’une architecture multi-agents (7 agents) :
    • Collecte & ingestion de données temps réel (GTFS-RT, SIRI Lite)
    • Nettoyage et normalisation des flux
    • Détection d’anomalies sur séries temporelles
    • Prise de décision automatisée
    • Génération de rapports intelligents via LLM
  • Algorithme de détection avancée :
    • Z-score contextualisé (baseline historique)
    • Analyse de dérive (trend / slope)
    • Persistance temporelle pour filtrer le bruit
  • Système de scoring intelligent :
    • Gravité opérationnelle
    • Impact utilisateur (trafic, heures de pointe)
    • Indice de confiance statistique
  • Pipeline data end-to-end :
    • Ingestion → Snowflake
    • Traitement → Python
    • Visualisation → Streamlit dashboard

Résultat : Détection des incidents plusieurs minutes avant les systèmes classiques.

🤖 AI Customer Support System (RAG + NLP)

Objectif : Automatiser et améliorer la gestion des tickets clients via IA.

  • Implémentation d’un pipeline RAG (Retrieval Augmented Generation)
  • Construction d’une base de connaissances vectorielle avec embeddings
  • Recherche sémantique pour retrouver les informations pertinentes
  • Analyse de sentiment pour priorisation automatique des tickets
  • Génération de réponses contextualisées via LLM
  • Architecture modulaire multi-composants

Résultat : Réduction du temps de traitement et amélioration de la qualité des réponses.
